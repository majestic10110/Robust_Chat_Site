Simple, sturdy radio chat.

Welcome to ROBUST CHAT, programmed by Artificial Intelligence with the help of Oliver Cohen M0OLI. It is a lightweight desktop program for amateur radio text messaging, built for Link500 and Teensy RPR TNC devices.

Fleet Manager

Create groups, whitelist callsigns, and filter traffic. Designed for net control and organised operations.

ACK Logic

Send/receive ACKs with configurable pause (default 12s). Message colour cues for sent, received, and ACKed.

GPS & Beacons

Quick “GPS SEND” with position format ready for on‑air exchange. Optional 15‑minute beacon timer.

Lightweight UI

Retro terminal aesthetic, large readable fonts (VT323), and keyboard shortcuts for fast operation.

Portable Setup

Zero servers. Just your radio + TNC. Works well for ad‑hoc field deployments and emergency comms.
